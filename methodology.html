Methodology

Dataset, Modelling Approach and Limitations

This page explains how the project was structured and why the model uses only information available after Game 2.

Dataset

330 completed best-of-seven playoff series.

The dataset covers NBA playoff series from 2004 to 2025. Each row represents one series and captures the Game 1 winner, Game 2 winner, final series winner, series state after Game 2, Game 2 margin, home/away status and playoff round.

Period 2004–2025 All included rounds were best-of-seven
Rows 330 One row per playoff series
Target 0 / 1 Whether Game 2 winner won the series
Validation 2026 Completed series used as out-of-sample test

Feature Design

Only information known after Game 2 was used.

This avoids data leakage. The model does not use final series result, series length, or other future information. The aim is to estimate probability from the situation available immediately after Game 2.

Variable Used? Reason
after_game2_state Yes Captures whether the Game 2 winner led 2-0 or tied the series 1-1.
game2_margin Yes Measures the strength of the Game 2 win.
game2_winner_home Yes Captures whether the Game 2 winner played at home or away.
round Yes Allows prediction to vary by playoff stage.
series_result No Future outcome. Using it would leak the answer into the model.
series_length No Known only after the series ends, so it is not valid for prediction after Game 2.

Limitations

What this model does not know

This analysis shows association, not causation. The model does not fully control for injuries, player availability, team strength, roster changes, coaching adjustments, matchup-specific factors, or betting market expectations. Later-round samples are smaller and the 2026 validation sample is based only on completed series available at the time of testing.

Future Work

The next upgrade is team-strength data.

A stronger model could add seed difference, regular-season win difference, Elo rating, net rating, injury indicators, or betting market probabilities. Those variables would help separate the Game 2 effect from team quality.
